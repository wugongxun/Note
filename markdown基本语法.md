# markdown基本语法

## 1.标题

```java 
# 一级标题

## 二级标题

### 三级标题

 ......
```

# 一级标题

## 二级标题

### 三级标题

 ......

## 2.字体

```java 
*这是倾斜的文字*
**这是加粗的文字**
***这是斜体加粗的文字***
~~这是加删除线的文字~~
==这是高亮的文字==
```

*这是倾斜的文字*

**这是加粗的文字**
***这是斜体加粗的文字***
~~这是加删除线的文字~~

==这是高亮的文字==

## 3.引用

```java
>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容
```

>这是引用的内容
>>这是引用的内容
>>
>>>>>>>>>>这是引用的内容

## 4.分割线

```java
---
----
***
*****
```

---
----
***
*****
